# PyMicroNanoFlow

Premium computational framework for:

- Unsteady micropolar hybrid nanofluid flow
- Curved surface transport analysis
- Heat transfer simulation
- Entropy generation analysis
- Stability-oriented computational studies

## Run locally

```bash
pip install -r requirements.txt
streamlit run pymicronanoflow_premium_app.py
```
